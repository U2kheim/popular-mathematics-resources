# Popular Mathematics Resources

A curated list of popular-science resources (websites, podcasts, books, articles) intended to support [Tokheim, I. (Unpublished). Popular science as a catalyst for intrinsic motivation in mathematics education [Doctoral thesis]. Western Norway University of Applied Sciences.]

How to update
- Edit data/resources.csv in this repository (click the file in GitHub, then the pencil icon to edit).
- Commit changes and they will appear on the published site.

View the interactive list
- Enable GitHub Pages in the repository Settings → Pages → Select branch: main (or default branch), folder: / (root).
- The site will be live at https://<your-github-username>.github.io/<repo-name>/ (may take a minute).

CSV format
- The CSV must have these headers (case-sensitive): Resource,Type,Topic,Description,Link
- Additional columns are allowed and will be shown in the table.
